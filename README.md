# 株式会社SCH コーポレートサイト

営業代行サービスの集客向け、女性らしいナチュラルデザインのコーポレートサイト（1ページ完結型）。
ピンク×ベージュのカラーパレットと多彩なスクロールアニメーションが特徴。

## 完成済み機能

- ヘッダー（スクロール追従・ガラス感のあるすりガラスデザイン・ハンバーガーメニュー）
- ヒーロー（浮遊バブルアニメーション・テキストフェードイン）
- 特徴3点バナー（丁寧な対応・成果・コミュニケーション）
- **ABOUT US**（会社メッセージ3ブロック＋会社情報テーブル）
- **サービス内容**（テレアポ・商談代行・新規開拓・戦略支援の4サービス）
- **選ばれる理由**（6つの理由カード・カウントアップアニメーション）
- **ご利用の流れ**（4ステップ・ウェーブアニメーション）
- CTAセクション（無料相談の誘導）
- お問い合わせフォーム（バリデーション・フェードイン送信完了・APIへのデータ保存）
- フッター

## アニメーション一覧
